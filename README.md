# APEX ASSISTANT
**Voice-Operated Workflow | CIP Course Project**

---

## Overview
**Apex Assistant** is a high-performance, voice-controlled desktop automation tool designed to streamline productivity. Inspired by the precision and speed of Formula 1 telemetry, this assistant acts as your digital "pit crew," allowing you to execute complex system commands via simple voice inputs.

Built for the **CIP** course, this project bridges the gap between natural language processing and system-level execution, ensuring you stay in the lead of your digital tasks.

## Key Features
* **Voice-Triggered Command Execution:** High-speed processing of intent to launch local applications.
* **Telemetry Control:** Hands-free volume management using integrated desktop automation.
* **Dynamic Search:** Seamlessly bridge the gap between spoken queries and YouTube results.
* **Fault-Tolerant Engine:** Includes a robust "Type-to-Command" fallback system to ensure your workflow never stalls.

## Technical Specifications
* **Core Language:** Python 3.x
* **NLP & Speech:** `spaCy`, `SpeechRecognition`, `pyttsx3`
* **Automation:** `pyautogui`, `webbrowser`, `os`
* **Environment:** Optimized for Windows 10/11
